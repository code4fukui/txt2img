# txt2img

make .env
```
OPENAI_API_KEY=****
```

create a single image
```sh
deno run -A https://code4fukui.github.io/txt2img/txt2img.js "koala wearing a strawberry hat"
```

create 17 variations of images
```sh
deno run -A https://code4fukui.github.io/txt2img/txt2img.js "koala wearing a strawberry hat"
```
